Scratch

Year 4 - 6

Scratch is a visual programming language that can be used by students, scholars, teachers, and parents to easily create games and provide a stepping stone to the more advanced world of computer programming.

We use Scratch to introduce children to programming and to teach fundamental computing concepts: algorithms, flow of control, repetition and selection. This is through solving a series of fun challenges to create animations, games and simulations.

Python

Year 7 - 9

Python is a widely used general-purpose, high-level programming language. Its design emphasises code readability, and is widely used in education due to its comparatively simple syntax.

We use Python to introduce children to syntax programming and a number of more advanced computing concepts: functions, procedures, recursion and logic. Students create programmes to draw geometric patterns and encrypting secret messages.
